# Set up instructions
1. Fork the repo
2. Clone your fork
3. Create a branch
4. Run `yarn`
5. Run `yarn start`
6. Make your changes and try to make the tests pass.
7. Commit with `git add` and `git commit`
8. Push your changes to your fork with `git push`
9. Create a pull request.
10. Iterate on the solution.
12. Get merged
